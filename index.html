Comandos de Git

  • git init
  • git status
  • git add [nombre de archivo] o bien -A
  • git commit -m "mensaje de identificación"
  • git log
  • git checkout [código sha] / master
  • git reset --soft / --hard

El concepto de Head

El Head es el commit donde nos encontramos actualmente, no importa si hemos hecho checkout

Ramas

Una rama es una línea de tiempo en nuestro código que nos sirve para experimentar, corregir errores, desarrollar nuevas características, etc.

  1. La rama Master es la rama principal y estable del proyecto
    • git branch
    • git branch [nombre de la nueva rama]
    • git branch -D [rama a eliminar]
  2. Las fusiones permiten crear un commit para unir dos ramas
    • git merge [rama que será absorbida]

Trabajando con Github

  • git clone [url del proyecto remoto]
  • git remote add origin [url del nombre del proyecto en Github
  • git remote remove origin
